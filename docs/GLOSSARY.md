# Glossary

## A

**API** - Application Programming Interface  
**Accessibility** - Making applications usable by everyone  
**Authentication** - Verifying user identity

## C

**Component** - Reusable UI element  
**CI/CD** - Continuous Integration/Continuous Deployment  
**Coverage** - Percentage of code tested

## E

**E2E** - End-to-End testing  
**ESLint** - JavaScript linter  
**Ethereum** - Blockchain platform

## F

**Feature Module** - Self-contained feature implementation  
**Form Validation** - Checking form input correctness

## H

**Hook** - React function for state/lifecycle  
**HOC** - Higher-Order Component

## I

**Integration Test** - Testing component interactions  
**Interface** - TypeScript type definition

## J

**Jest** - JavaScript testing framework  
**JSX** - JavaScript XML syntax

## N

**Next.js** - React framework  
**NFT** - Non-Fungible Token

## P

**Prisma** - Database ORM  
**Props** - Component properties  
**Playwright** - E2E testing tool

## R

**React** - UI library  
**RTL** - React Testing Library

## S

**SIWE** - Sign-In With Ethereum  
**SSR** - Server-Side Rendering  
**Storybook** - Component development tool

## T

**TypeScript** - Typed JavaScript  
**TDD** - Test-Driven Development  
**Tailwind** - Utility-first CSS

## U

**Unit Test** - Testing individual functions  
**Utility** - Helper function

## V

**Validation** - Checking data correctness  
**Viem** - TypeScript Ethereum library

## W

**Wagmi** - React Hooks for Ethereum  
**Web3** - Decentralized web  
**Wallet** - Cryptocurrency wallet
